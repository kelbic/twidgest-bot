## 1. Что такое Сервис

TwidgestBot — Telegram-бот для автоматизации новостных каналов. Сервис:

- Получает публичные твиты из X (Twitter) с указанных пользователем аккаунтов.
- Адаптирует и переводит их на русский язык с помощью больших языковых моделей.
- Публикует адаптированный контент в Telegram-каналы пользователя.
