# Политика конфиденциальности TwidgestBot

**Последнее обновление:** 26 апреля 2026 г.

Мы (далее — «Сервис», TwidgestBot) обрабатываем минимально необходимый объём данных для работы Telegram-бота, помогающего автоматизировать публикацию контента из X (Twitter) в Telegram-каналы.

## 1. Какие данные мы собираем

При использовании Сервиса мы получаем и храним:

- **Идентификатор Telegram-пользователя** (`tg_user_id`) — числовой ID, выдаваемый Telegram. Не содержит ФИО или контактных данных.
- **Telegram username** (если установлен) — публичный псевдоним пользователя в Telegram.
- **Конфигурация каналов**: список X-аккаунтов для мониторинга, ID Telegram-каналов для постинга, выбранные настройки фильтров, режим работы (digest/single/hybrid).
- **Историю платежей через Telegram Stars**: сумма, тариф, идентификатор транзакции от Telegram. Данные банковских карт **не обрабатываются Сервисом** — все платежи проходят через Telegram.
- **Технические логи**: IP-адрес запросов **не сохраняется**; в логах присутствуют ID пользователей, ID каналов, тексты ошибок для отладки.

## 2. Какие данные мы НЕ собираем

- ФИО, дата рождения, паспортные данные.
- Адрес электронной почты, телефон, физический адрес.
- Биометрические данные.
- Местоположение пользователя.
- Содержимое личных переписок пользователя в Telegram, кроме сообщений, направленных непосредственно боту.

## 3. Как мы используем данные

Собранные данные используются исключительно для:

- Технической работы Сервиса (получение твитов, генерация постов, публикация в каналы).
- Применения тарифных лимитов и обработки платежей.
- Диагностики и устранения сбоев.
- Уведомлений пользователю о состоянии его каналов.

**Мы НЕ продаём, НЕ передаём третьим лицам и НЕ используем данные для рекламы.**

## 4. Третьи стороны (Sub-processors)

При работе Сервис обращается к следующим внешним сервисам:

| Сервис | Назначение | Какие данные передаются |
|---|---|---|
| Telegram Bot API | Получение/отправка сообщений | tg_user_id, тексты сообщений |
| twitterapi.io | Получение твитов из X | username публичных X-аккаунтов |
| OpenRouter (Anthropic Claude / Llama) | Генерация и адаптация текста | тексты твитов, описания каналов |
| Unsplash | Подбор иллюстраций | ключевые слова на английском |

Условия обработки данных у этих сервисов регулируются их собственными политиками конфиденциальности.

## 5. Хранение и удаление

- Данные хранятся в зашифрованной БД на VPS-сервере.
- При удалении канала через `/deletechannel` все связанные источники, очередь дайджеста и логи отказов удаляются каскадно.
- При запросе на полное удаление аккаунта (направленном через [GitHub Issues](https://github.com/kelbic/twidgest-bot/issues) или Telegram-сообщение администратору) все ваши данные удаляются в течение 7 рабочих дней.

## 6. Права пользователя

Вы имеете право:

- Получить копию ваших данных, обратившись к администратору.
- Запросить корректировку или удаление данных.
- Прекратить использование Сервиса в любой момент через `/deletechannel` для каждого канала и блокировку бота в Telegram.

## 7. Безопасность

- API-ключи и секреты хранятся в защищённом конфиге сервера, не доступном публично.
- Платежи проходят через инфраструктуру Telegram — Сервис не имеет доступа к данным карт.
